# Axios

`axios` 是一个基于 `Promise` 用于`浏览器`和 `nodejs` 的 `HTTP` 客户端，本质上也是对原生 `XHR` 的封装，只不过它是 `Promise` 的实现版本，符合最新的ES规范，有以下特点：

- 从浏览器中创建 `XMLHttpRequests`
- 从 `node.js` 创建 `http` 请求
- 支持 `Promise API`
- 拦截请求和响应
- 转换请求数据和响应数据
- 取消请求
- 自动转换 `JSON` 数据
- 客户端支持防御 `XSRF`
